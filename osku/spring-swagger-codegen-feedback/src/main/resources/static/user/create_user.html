Tervetuloa!

OSKU-palautesovelluksella kerätään käyttäjiltä (esim. asukkaat) palautetta kiinteistöjen olosuhteista, kuten lämpötilasta ja ilmanlaadusta. Kerättävät tiedot tulevat Tampereen Yliopiston KIEMI-hankeen käyttöön. Palautetta käytetään yhdessä muun kiinteistöstä kerättävän tiedon (esim. energiankulutus, lämpötilamittaukset) kanssa kiinteistöjen energiatehokkuutta sekä olosuhteita parantavien menetelmien ja teknisten ratkaisujen kehittämiseen.

Luo uusi käyttäjätunnus syöttämällä haluamasi käyttäjätunnus ja salasana alla oleviin kenttiin. Asuntokohtaisten käyttäjätunnusten luominen on suositeltua (esim. käyttäjätunnuksella paratiisitie13asunto1), mutta voit myös halutessasi luoda nimimerkillä anonyymin käyttäjätunnuksen. Salasanan tulee olla vähintään merkkiä pitkä. Huomioithan, että käyttäjätunnusta (tai salasanaa) ei voi muuttaa tai poistaa myöhemmin.
